# Xawars RNG - Pending Features

## Content Creation Features

1. **Session Stats Panel** - Show total operators used, most common role, longest session, etc.
2. **Streak Counter** - Track how many matches in a row you've won/kept the same operator
3. **Export Session** - Download session summary as image or text for video descriptions
4. **Operator Filter** - Exclude operators you don't want to play (e.g., "no drones")
5. **Game Mode Selector** - Support for different game modes (Bomb, Secure Area, Hostage)
6. **Round Tracker** - Track wins/losses per operator
7. **Timer** - Session duration timer, time per operator
8. **Custom Loadout Rules** - "Pistol only", "No gadgets", etc.
9. **Thumbnail Overlay** - Toggle an OBS-ready overlay showing current operator info
10. **Voice Line Picker** - Add operator-specific quotes/sounds on deploy
11. **Spin Animation** - More dramatic/visual roll animation for video content
12. **Clip Highlights** - Mark "clip moments" during session to review later
13. **Operator Audio Quotes** - Play operator voice lines on deploy ("Deploying now!" etc.)
14. **Quick Share Button** - One-click copy/paste current roll to clipboard for sharing
15. **OBS Overlay Mode** - Even more minimal mode specifically for streaming (hide everything except operator name + role)
16. **Spin Animation Controls** - Adjust speed/intensity of roll animation
17. **Full Team Generator** - Generate 5 operators at once (full attack/defense lineup)
18. **Operator Favorites** - Mark operators as favorites for weighted random
19. **Custom Keyboard Shortcuts** - Hotkeys for roll, kill, death, reroll
20. **Auto-Roll on Target Complete** - Option to auto-reroll when operator target is reached
21. **Operator Usage History** - Show which operators you've used most this session
22. **K/D per Operator** - Track kill/death ratio per specific operator
23. **Session Duration Timer** - Auto-tracking how long you've been playing
24. **Multiple Themes** - Different color schemes beyond yellow/black
25. **Custom Sound Packs** - Different sound themes (Sci-fi, Retro, etc.)
